    # Native Ad (Manual)

    Follow these steps to build a native ad layout and then load it.

    Step 1: Create Native Ad Layout
    Step 2: Initialize Configuration
    Step 3: Create AdapterHelper
    Step 4: Create MoPubNative
    Optional: Set Category
    Step 5: Create MoPubNativeNetworkListener
    Step 6: Create ViewBinder
    Step 7: Render ViewBinder and Register Renderer
    Step 8: Load Ads
    Step 9: Get Ad View

    {% hint style="warning" %} Note: If you are integrating multiple third-party ad networks like Facebook and Pangle, you must set up, create, and register the appropriate renderer for each ad network. Remember to register the default MoPubStaticNativeAdRenderer last. The order of renderer registering is crucial. Wrong ordering might lead to ad malfunction which affects your revenue.
    Refer to MoPub Integrate Third-Party Ad Networks (opens new window) for more detail. {% endhint %}

    # Step 9: Get Ad View

    Get the ad view in the callback function of onNativeLoad .

    {% tabs %} {% tab title="Kotlin" %}

    override fun onNativeLoad(nativeAd: NativeAd) {

            val adView =
                adapterHelper.getAdView(
                    null,
                    null,
                    nativeAd,
                    ViewBinder.Builder(0).build()
                )
                
            // It a parent view
            viewBinding.linearLayout.addView(adView)

}
