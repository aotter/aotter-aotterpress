    # Tracker

    Besides sending user's info to AotterTrek for a better ad experience to your users, we provide an advanced service Tracker. Base on User , Entity and Action data, our system will use these data for user profiling. AotterTrek groups these users with similar behavior and send them personalized ads that improve advertising relevance for users and increases revenue for you.

    Step 1: Initialize Tracker Service
    Step 2: Create Entity / User Object Instance
    Step 3: Choose Action Types
    Step 4: Send Tracker

    # Step 1: Initialize Tracker Service

    Notice that you should initialize AotterTrek.TrackerService before using Tracker, otherwise, an exception will be thrown.

    # Step 2: Create Entity / User Object Instance

    Entity constructor parameter

    The following table shows entity constructor parameters which including basic information about the entity such us id , title and type and so on.

    Name Type description
    id String
    title String
    type String

    The Parameter type of the type:

    EntityType.POST.type

    EntityType.PLACE.type

    EntityType.GAME.type

    EntityType.MUSIC.type

    EntityType.VIDEO.type

    EntityType.MERCHANT.type

    EntityType.ITEM.type

    EntityType.UNKNOWN.type

    url String
    categories List<String>
    tags List<String>
    meta JsonObject

    val jsonObject = JsonObject() //Kotlin

    JsonObject jsonObject = new JsonObject() // Java

    Options:

    jsonObject.addProperty(TrekDataKey.REFERENCE,VALUE) //Ex: "dips" jsonObject.addProperty(TrekDataKey.PUBLISHED_DATE,VALUE)//Ex:1438090882490L jsonObject.addProperty(TrekDataKey.IMG,VALUE)//Ex:"http://pnn.aotter.net/Media/show/cna.jpg"

    jsonObject.addProperty(TrekDataKey.AUTHOR,VALUE)//Ex: "skybear"

    jsonObject.addProperty (TrekDataKey.ADDRESS,VALUE)//Ex: "105 台北市松山區南京東路四段 2 號"

    jsonObject.addProperty(TrekDataKey.LAT, VALUE)//Ex: 25.0463684

    jsonObject.addProperty(TrekDataKey.LNG, VALUE)//Ex: 121.5501565

    User constructor parameter

    The following table shows the user birthday , email and gender and so on.

    Name Type description
    birthday String Ex: "1999/10/10"
    email String Ex: "a111111@gmail.com"
    fbId String
    gender String "M" for male, "F" for female
    phone String Ex: "09XXXXXXXXX"
    meta JsonObject

    val jsonObject = JsonObject() //Kotlin

    JsonObject jsonObject = new JsonObject() // Java

    //add you want to define key and value

    jsonObject.add(KEY, VALUE)
    jsonObject.add(KEY, VALUE)
    //or
    jsonObject.addProperty(KEY, VALUE)
    jsonObject.addProperty(KEY, VALUE)

    Ex:
    jsonObject.addProperty("publishedDate", 123456L)

    jsonObject.addProperty("reference", "aotter")
